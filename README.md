# fp01 - Website

## Português (Portugal)

### Descrição Geral
O projeto "fp01" é um **formulário HTML minimalista** concebido para demonstrar a recolha de dados básicos do utilizador. Serve como um ponto de partida simples para projetos web que requerem interação de formulário.

---

### Funcionalidades Principais
1. **Recolha de Dados Essenciais**:
   - Campos de entrada para **Nome**, **Password** e **Idade**.
   - O campo "Password" utiliza o tipo `password` para ocultar a entrada.

2. **Submissão de Formulário**:
   - O formulário está configurado para enviar os dados para um *endpoint* de servidor (`resultado.php`) utilizando o método **POST**.

3. **Estrutura Limpa**:
   - Código HTML5 básico e sem estilos, focado puramente na marcação estrutural.

---

### Estrutura do Ficheiro
- **`form.html`**:
  - Contém a estrutura completa do formulário.
  - Utiliza a *tag* `<form>` com `action="resultado.php"` e `method="POST"`.
  - Inclui *tags* `<label>` para acessibilidade e *tags* `<input>` para os campos de dados.
  - O botão de submissão é um `<button type="submit">`.

---

### Como Utilizar
1. **Abrir o Ficheiro**:
   - Localize o ficheiro `form.html` no seu sistema.
   - Abra-o diretamente no seu navegador web preferido (Chrome, Firefox, etc.).

   - Clique no botão **Enviar**.

> **Nota:** Para que a submissão seja processada com sucesso, é necessário que um servidor web (como Apache ou Nginx) esteja a correr e que o ficheiro `resultado.php` exista e esteja configurado para receber e processar os dados POST.
